# ContextQB Methodology

Architectural thinking for AI-native builders.

This repository contains the **methodology content** — principles, playbooks, audits, and prompts that teach systems thinking for AI-assisted software development.

## What's here

```
content/
  principles/    ← Core architecture principles
  playbooks/     ← Step-by-step workflows
  audits/        ← Agent audit templates
  prompts/       ← Reusable agent prompts
format/
  SPEC.md        ← The context.qb file format specification
  examples/      ← Example context.qb files
docs/
  mcp-setup.md   ← How to connect your AI agent to the methodology
```

## How to use it

### Read on the web

Browse the full methodology with rendered formatting at **[contextqb.com](https://contextqb.com)**.

### Connect your AI agent

The ContextQB MCP server gives your AI agent direct access to all principles, playbooks, audits, and prompts. See [docs/mcp-setup.md](docs/mcp-setup.md) for configuration.
