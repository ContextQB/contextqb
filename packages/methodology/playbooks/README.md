# @contextqb/playbooks

ContextQB playbooks and audit templates. **Content only — no code.**

Two subdirectories:

- `playbooks/` — Step-by-step workflows for common tasks (planning a feature, cleaning up a repo, writing agent instructions).
- `audits/` — Structured templates for asking an agent to evaluate a system end-to-end.

Both validated by `@contextqb/content`. Schemas live in [`packages/content/src/schema.ts`](../content/src/schema.ts).
